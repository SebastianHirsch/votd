votd
an api and shell tool for a bible verse of the day
source | rubygems | docs
$ gem install votd
require 'votd'

# See our docs for more supported APIs
votd = Votd::BibleGateway.new

votd.reference # e.g. John 3:16
votd.text      # e.g. "For God so loved the world..."
votd.date      # e.g. 2012-03-30
votd.version   # e.g. NIV
votd.copyright # e.g. Copyright ©...

votd.to_html   # e.g. <p class="votd">For God so loved the world...>/p>

# The output of the HTML looks like the following:
<p class="votd-text">For by grace you are saved through faith...
 it is not from works, so that no one can boast.</p>
<p>
<span class="votd-reference"><strong>Ephesians 2:8-9</strong></span>
<span class="votd-version"><em>(NETBible)</em></span>
</p>
