votd
an api and shell tool for a bible verse of the day
source | rubygems | docs

Installation is as easy as one-two-three:

$ gem install votd

Then just add the template:

    require 'votd'

    verse = Votd::NetBible.new
    
    my_sites_verse = "
    
    Verse of the day for #{verse.date}
    
    #{verse.text}
    -- #{verse.reference}, #{verse.version}"
    
    # verse.date      -- current date"
    # verse.text      -- the verse itself
    # verse.reference -- bible reference (e.g. John 3:16)
    # verse.version   -- bible version (e.g. King James)

Then simply add the template into your ERB HTML:

      <%= my_sites_verse %>

      <!-- Or use the easier... -->

      <%= verse.to_html %>

      <!-- The built-in to_html
 method turns the verse into html
 code. -->

The to_html will always output: 

    <p> class="votd-text"For by grace you are saved through faith... 
it is not from works, 
so that no one can boast.</p>
<p>
    <span class="votd-reference"><strong>Ephesians 2:8-9</strong></span>
    <span class="votd-version"><em>(NETBible)</em></span>
</p>
